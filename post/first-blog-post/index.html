First blog post

2017/05/28

And after a few days of troubleshooting, I have finally succeeded in launching my first website with Hugo, hurray!

My current setup is GitHub Pages + Hugo + Blogdown as recommended in this excellent tutorial by Amber Thomas. This allows me to create rich web content by utilizing Rmarkdown files with the blogdown package in Rstudio. This smoothly integrates with my personal data analysis projects that are mainly done in the R statistical programming language. As an Rmarkdown file is essentially an extended version of markdown it is also still possible to include plain html markup, and create beautiful mathematical equations with LaTeX.

I chose to use Hugo because it solves many of the shortcommings of competitors like Jekyll and Wordpress. The main advantage of Hugo is that it produces static websites. The idea of a static website is that websites are viewed far more often than they are edited, hence the building is only done when the content is created. Another important feature of Hugo are the hundreds of themes that are available through the theme gallery. Most themes are a bit basic, however, there are some very great ones to choose from. I decided to use the beautiful hugo theme. Out of the box, this theme comes with lots of functionality. For example, there is the possibility to integrate Disqus, a dynamic forum-like framework, and Google Analytics, which is a tool that gives detailed insight in the viewers of a website.

My intention is to use the blog part of this webpage to publish meta information, that means, things concerned with my personal life, and or the organization of this webpage.
