Fist blog post

Finally, after a few days of troubleshooting, I have succeeded in launching my first website with Hugo!

My current setup is GitHub Pages + Hugo + Blogdown as described in this excellent tutorial by Amber Thomas. This allows me to create rich web content by utilizing Rmarkdown files with the blogdown package in Rstudio.

This is a R chunk. I can for example add two numbers like this:

1 + 3 
## [1] 4

As this Rmarkdown file is essentially an extended version of markdown it is also still possible to use plain html markup, and create beautiful mathematical equations in LaTeX: \[ e^{\pi i} -1 = 0 \] I chose to use Hugo because I saw that recently many websites had migrated to it. The main feature of Hugo is that it produces static websites. The idea of a static website is that websites are viewed far more often than they are edited, hence the building is only done when the content is created. Another important feature of Hugo are the hundreds of themes that are available through the theme gallery. Most themes are not that interesting, however, there are some very great ones to choose from. I decided to use the beautiful hugo theme.
